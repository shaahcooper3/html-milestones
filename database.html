THIS IS YOUR TAGS SECTION:

head: Creates heading
body: Creates body
title: Creates new title
h1: Creates new heading
marquee: Moves text across row on screen
!-- --: Comments out lines (VIEW EXAMPLE IN VS CODE)
br: Prints on a new line (DOES NOT HAVE A END TAG)
strong: Puts element in bold
em: Puts element in emphasis
a: Anchor tag is used to define the beginning and end of a hypertext link.

THIS IS YOUR ATTRIBUTES SECTION:

href: Hyperlink reference to a linked source.
id: Identifies an element
class: Classifies an element- Specifies one or more class names for an HTML element
src: Specifies a source for embeddable content
